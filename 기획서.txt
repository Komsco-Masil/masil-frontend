- 목차

## 1. 서비스 개요

| 항목 | 내용 |
| --- | --- |
| 서비스명 | 마실 |
| 팀명 | 갈래유 |
| 서비스 유형 | 지역사랑상품권 가맹점 기반 소상공인 전용 SNS 커뮤니티 플랫폼 |
| 핵심 콘셉트 | 지역사랑상품권 가맹점으로 인증된 가게들이 모여 홍보, 정보 공유, 모임 형성, 쪽지 소통을 할 수 있는 지역 기반 커뮤니티 |
| 참고 서비스 | 에브리타임, 블라인드, 당근 |
| 핵심 가치 | 인증된 지역 가게만 참여하는 신뢰 기반 커뮤니티 |

---

## 2. 문제 정의

| 대상 | 문제점 |
| --- | --- |
| 소상공인 | 가게 홍보 채널이 부족하고, 같은 지역 상권 정보를 공유할 공간이 부족함 |
| 지역 주민 | 지역사랑상품권 사용 가능 가게를 찾기 어렵고, 동네 기반 모임을 만들기 불편함 |
| 지자체/공공기관 | 지역사랑상품권 데이터가 단순 조회에 머무르고 실제 소비 활성화로 이어지기 어려움 |

---

## 3. 서비스 목표

- 지역사랑상품권 가맹점 데이터를 활용하여 신뢰도 높은 가맹점 인증 시스템을 구축합니다.
- 소상공인이 직접 세일, 이벤트, 신메뉴, 매장 소식을 홍보할 수 있는 공간을 제공합니다.
- 지역 주민이 “같이 여기 갈 사람”처럼 동네 기반 모임을 만들 수 있도록 지원합니다.
- 소상공인끼리 운영 정보, 지역 상권 정보, 정책 정보를 공유할 수 있는 커뮤니티를 제공합니다.
- 지역사랑상품권 사용처를 단순히 보여주는 것을 넘어 실제 방문과 소비로 연결합니다.

---

## 4. 주요 사용자

| 사용자 유형 | 설명 | 주요 기능 |
| --- | --- | --- |
| Admin 관리자 | 서비스 운영자 | 회원 관리, 게시글 관리, 신고 처리, 배지 관리 |
| User A 사장 | 지역사랑상품권 가맹점 대표자 | 가게 인증, 가게 홍보, 행사 등록, 소상공인 게시판 이용 |
| User B 직원 | 인증 가맹점 소속 직원 | 가게 소식 작성, 게시판 참여, 쪽지 이용 |
| User C 일반 손님 | 지역 주민 및 소비자 | 가게 정보 확인, 모임 게시판 이용, 쪽지 이용 |

## 5. 핵심 기능

## 5-1. 가맹점 인증 회원가입

지역사랑상품권 가맹점 공공데이터를 활용하여 회원가입 시 가게 정보를 검증합니다.

| 기능 | 설명 |
| --- | --- |
| 가맹점 정보 입력 | 사업자 또는 가게명, 지역, 주소 등 기본 정보 입력 |
| 공공데이터 대조 | 지역사랑상품권 가맹점 데이터와 입력 정보 비교 |
| 인증 완료 | 데이터와 일치하는 경우 커뮤니티 이용 권한 부여 |
| 권한 구분 | 사장, 직원, 일반 손님 등 사용자 유형별 권한 분리 |

---

## 5-2. 가게 홍보 게시판

소상공인이 자신의 가게 소식을 직접 올릴 수 있는 공간입니다.

| 기능 | 설명 |
| --- | --- |
| 세일 홍보 | 할인 행사, 기간 한정 이벤트 등록 |
| 신메뉴 홍보 | 신메뉴, 시즌 메뉴, 신규 상품 소개 |
| 매장 소식 | 휴무일, 영업시간 변경, 예약 안내 등 공지 |
| 지역사랑상품권 사용 가능 표시 | 해당 가게가 지역사랑상품권 가맹점임을 강조 |

예시 게시글 제목)

- 오늘만 10% 할인합니다.
- 지역사랑상품권 결제 가능 매장입니다.
- 이번 주 신메뉴 출시했습니다.
- 비 오는 날 방문 고객 서비스 제공

---

## 5-3. 소상공인 익명 게시판

블라인드처럼 소상공인끼리 익명으로 정보를 나눌 수 있는 공간입니다.

| 기능 | 설명 |
| --- | --- |
| 익명 글 작성 | 가게명 노출 없이 고민, 정보, 질문 작성 |
| 운영 노하우 공유 | 알바 관리, 재고 관리, 손님 응대, 매출 고민 공유 |
| 정책 정보 공유 | 지원금, 지자체 정책, 상품권 운영 정보 공유 |
| 댓글 소통 | 다른 소상공인과 실시간 의견 교환 |

예시 게시글 제목

- 요즘 평일 매출 다들 어떤가요?
- 알바 급하게 구할 때 어디가 제일 효과 좋나요?
- 지역사랑상품권 결제 손님이 늘었나요?
- 배달앱 수수료 때문에 고민입니다.

---

## 5-4. 유저 모임 게시판

당근처럼 지역 주민들이 특정 가게 방문 모임을 만들 수 있는 기능입니다.

| 기능 | 설명 |
| --- | --- |
| 모임 글 작성 | “같이 여기 갈 사람” 형태의 모임 모집 |
| 장소 연결 | 지역사랑상품권 사용 가능 가게와 모임 연결 |
| 참여 신청 | 관심 있는 사용자가 모임에 참여 |
| 쪽지 연결 | 모임 개설자와 참여자가 쪽지로 세부 일정 조율 |

예시 게시글 제목

- 오늘 저녁에 여기 같이 갈 사람 있나요?
- 주말에 동네 카페 투어하실 분
- 상품권 되는 맛집 같이 가요
- 혼밥하기 애매해서 같이 점심 먹을 분

---

## 5-5. 핫게 / 인기 게시글

활동량이 높은 게시글을 모아 보여주는 기능입니다.

| 기능 | 설명 |
| --- | --- |
| 인기 게시글 노출 | 조회수, 댓글 수, 좋아요 수 기준으로 인기글 선정 |
| 지역별 핫글 | 같은 지역 내 인기 게시글 확인 |
| 홍보 효과 강화 | 반응 좋은 가게 홍보글이 더 많이 노출될 수 있음 |

---

## 5-6. 쪽지 기능

사용자 간 개인적인 소통을 지원하는 기능입니다.

| 기능 | 설명 |
| --- | --- |
| 1:1 쪽지 | 사용자 간 개별 메시지 전송 |
| 모임 조율 | 모임 시간, 장소, 인원 확인 |
| 가게 문의 | 가게 이벤트나 이용 가능 여부 문의 |
| 신고 기능 | 부적절한 쪽지 신고 가능 |

---

## 5-7. 배지 시스템

활동량과 기여도를 기반으로 사용자에게 배지를 부여합니다.

| 배지 예시 | 설명 |
| --- | --- |
| 이달의 홍보왕 | 홍보 게시글 반응이 가장 높은 가게 |
| 소통왕 | 댓글과 답변 활동이 활발한 사용자 |
| 동네 연결왕 | 모임을 많이 개설하거나 참여한 사용자 |
| 정보 공유왕 | 소상공인 게시판에서 유용한 정보를 많이 공유한 사용자 |
